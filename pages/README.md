# 测试页面说明

本文件夹包含按照设计图（测试页草图.pdf）创建的所有测试页面。

## 📁 文件列表

### PDF相关页面（六宫格布局）
- `pdf-home.html` - PDF首页，包含六宫格导航（界面剂、自流平、胶粘剂、木地板相关、WOLFF、Pajarito）
- `pdf-category.html` - PDF二级页，显示产品列表（2列网格布局）
- `pdf-detail.html` - PDF三级页，产品详情和PDF下载页面

### 视频相关页面（三宫格布局）
- `video-home.html` - 视频首页，包含三宫格导航（弹性地板、木地板、工具）
- `video-category.html` - 视频二级页，显示视频分类列表
- `video-detail.html` - 视频三级页，视频播放列表页面

## 🎨 设计特点

### 通用元素
- **状态栏**：显示时间"10:40"和关闭按钮"X"
- **品牌头部**：UZIN优成 + uzin.asia
- **轮播图**：3张图片自动轮播切换（3秒间隔）
- **底部Logo**：UZIN品牌标识（仅首页）

### PDF页面特色
- **六宫格导航**：采用不同颜色的圆形图标
  - 蓝色：界面剂、自流平、胶粘剂
  - 黄色：木地板相关
  - 青色：WOLFF
  - 红色：Pajarito
- **产品列表**：2列网格，包含产品图片、型号和中文名称
- **下载页面**：大红色PDF图标 + 橙色下载按钮

### 视频页面特色
- **三宫格导航**：不同形状的彩色图标
  - 蓝色多边形：弹性地板
  - 黄色圆形：木地板
  - 青色形状：工具
- **蓝色标题栏**：类目和页面标题采用渐变蓝色背景
- **视频播放**：支持横版和竖版视频，白色圆形播放按钮

所有资源文件相对于项目根目录：

- 视频类目图：`../assets/images/video/categories/`

## 🚀 如何使用

1. 在浏览器中打开项目根目录的 `test-index.html`
2. 点击相应的链接即可查看各个测试页面
3. 所有页面完全按照设计图还原，像素级精确

## 📱 响应式支持

- 移动端：优化的触摸交互和布局
- 桌面端：最大宽度768px，居中显示
- 小屏设备（≤360px）：图标和文字自动缩小

## 🎯 页面流程

### PDF流程
首页（六宫格） → 产品列表（2列） → 产品详情（下载页）

### 视频流程
首页（三宫格） → 类目列表（2×2网格） → 视频播放列表
